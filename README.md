
# Headline app challenge

Live: https://headlines-blue.vercel.app/

## Requirements
- Node LTS 16.x.x
- NPM

## Run the app
Navigate to the project folder and run the following command to install the dependencies
```bash
npm install
```

Then, run the following command to start the app
```bash
npm run dev
```
